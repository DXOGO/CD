## Trabalho 1 Computação Distribuída 2020/2021

O objectivo é desenvolver um sistema de chat básico. Os clientes ligam-se ao servidor através de uma socket TCP e enviam mensagens em formato JSON. O servidor recebe as mensagens dos clientes e propaga as mensagens para os restantes clientes ativos.
